# Proffy

## Requisitos

#### Instalação Backend 

```
yarn add install
```
```
yarn start
```

#### Instação Frontend
```
yarn add install

```
```
yarn start
```

### Instalação Mobile

```
yarn add install
```
```
yarn start
```

#### Opcional
##### Para testar o App no mobile
- Baixar o App Expo no App Store ou no Google Play.
 ```
yarn global add expo-cli
```
```
yarn start
```
